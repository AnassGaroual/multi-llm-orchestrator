# Security Policy

## Reporting a Vulnerability
If you find a vulnerability, please report it via [GitHub Security Advisories](https://github.com/AnassGaroual/multi-llm-orchestrator/security/advisories).

**Do NOT open a public issue for security vulnerabilities.**

## Security Measures

This project implements:
- ✅ Automated security scanning (CodeQL, Trivy, OWASP)
- ✅ Dependency vulnerability monitoring
- ✅ Secret detection (Gitleaks)
- ✅ Container image signing (Cosign)
- ✅ SBOM generation

## Security Updates

Security patches are released as soon as possible after disclosure.
